Goal icon expansion — 35 candidate shapes

Every goal icon in the app has two forms. Basic is the solid shape: a single fill of the goal color with a crisp outline, legible at node size. SMART is the same geometry brought to life: secondary-color detailing masked inside the silhouette, a looping animation on top, and a solid core dot — the shape "activates" when a goal is fully specified.

Every SMART preview below runs its own bespoke loop, in the spirit of the circle's rotating globe, the square's fractal zoom, and the triangle's triforce fill — perimeter traces, ratchets, drains, glints, sheens, heartbeats. The effect and its timing are specified on each card.

Current set — for reference

The eight shipped shapes from ICON_SHAPES + GoalIcon.jsx. In the app, circle, square, triangle, and the twelve-point star carry bespoke loops (globe meridians, fractal zoom, triforce fill, counter-rotating squares); the rest currently use the generic ripple tunnel shown here.

Adding a shape: each candidate needs (1) a full geometry in the 100×100 viewBox added to GoalIcon.jsx and AnimatedGoalIcon.jsx, (2) an entry in ICON_SHAPES in goalCharacteristics.js using the slug shown on its card, and (3) its bespoke SMART loop ported into AnimatedGoalIcon.jsx — every loop on this sheet is plain CSS keyframes on masked SVG groups, the same technique the component already uses, so each one ports directly. Until a loop is ported, the generic ripple tunnel applies automatically. All previews inherit the palette from design-tokens.css, so both themes are covered, and reduced-motion falls back to the static concentric detailing.
